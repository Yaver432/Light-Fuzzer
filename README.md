### 🚀 How to Use

## 1. Install dependencies
```bash 
    pip install -r requirements.txt
```
## 2. Start the local server
```bash
        python -m http.server 8000
```
## 3. Open in browser

        Go to:
```bash
        http://localhost:8000
```
## 4. Run the fuzzer
```bash
        python fuzzer.py
```
